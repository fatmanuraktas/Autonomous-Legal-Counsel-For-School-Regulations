# ⚖️ Autonomous Legal Counsel & Hybrid RAG Assistant (v2.5)

An advanced AI-driven legal assistant designed to synthesize local university regulations with real-time Turkish laws. This project utilizes a **Hybrid Reasoning Architecture**, combining local vector storage (ChromaDB) with live web grounding via the **Gemini 2.5 Flash** engine.

## 🚀 Key Features

* **Hybrid RAG Architecture:** Seamlessly integrates static local PDF data (regulations/bylaws) with dynamic web data.
* **Google Search Grounding:** Leverages 500 RPD (Requests Per Day) free tier grounding to fetch up-to-date Turkish Penal Code (TCK) and Supreme Court precedents.
* **Adversarial AI (Anti-Thesis):** Includes a specialized mode to critique generated legal petitions, identifying procedural vulnerabilities before submission.
* **Security & Privacy First:** Implements a "decoupled" data strategy where sensitive vector databases are stored locally and excluded from version control (Git).
* **Automated Petition Drafting:** Generates formal legal documents based on analyzed context.

## 🏗️ System Architecture



1.  **Ingestion:** Local PDFs are parsed and chunked using `RecursiveCharacterTextSplitter`.
2.  **Embedding:** Text segments are vectorized using `sentence-transformers` and stored in a local **ChromaDB**.
3.  **Reasoning:** The **Gemini 2.5 Flash** model acts as the central brain, executing a "Thinking" process to decide whether to use local data or perform a Google Search.
4.  **Synthesis:** The final legal advice is generated by merging university-specific rules with national laws.

## 🛠️ Tech Stack

* **Model:** Gemini 2.5 Flash (v1 SDK)
* **Orchestration:** LangChain
* **Vector DB:** ChromaDB
* **Embeddings:** HuggingFace (paraphrase-multilingual-MiniLM-L12-v2)
* **Frontend:** Streamlit
* **Environment:** Linux (Pop!_OS)
